# Next.js, Supabase, and Vercel stack

Family App will use Next.js with TypeScript for the responsive web app, Vercel for hosting and scheduled cron endpoints, Supabase Postgres/Auth for hosted persistence and Parent authentication, Drizzle for schema and migrations, and Tailwind CSS with shadcn/ui and lucide-react for the interface. This keeps the app deployable on the go from a single TypeScript codebase while giving the household workflows a relational database, server-side authorization, and a direct path to read-only Apple Calendar sync.
